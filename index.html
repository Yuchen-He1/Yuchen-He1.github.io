Hi! I am a second-year M.S. student in Computer Engineering at the University of California, Santa Barbara (UCSB), advised by Prof. Jonathan Balkind and Prof. Juan Luis Aragón.

My research interests lie in computer architecture, hardware/software co-design, memory hierarchy, and address translation for graph workloads.

Research

My research focuses on memory hierarchy and HW/SW co-design for domain-specific graph workloads. I am currently working on:

  • TLB & address translation optimization for graph workloads using ChampSim
  • Oracle-based STLB hot-page pinning and prototype L3-TLB design
  • Static analysis for SoC data movement (Matchmaker project with Balkind Lab)
  • DLP extraction in RISC-V binaries for general-purpose HW/SW co-design

Current Projects

  • Pinning Hot Pages to the STLB for Fast Address Translation in Graph Applications (with Balkind & Aragón, 2025– )
  • Static Analyzer for Protocol Compatibility Checking in Heterogeneous SoCs (Balkind Lab, 2025)
  • Data-Level Parallelism Pattern Extraction in Program Inner Loops (B.S. Thesis, MUST, 2022–2024)
